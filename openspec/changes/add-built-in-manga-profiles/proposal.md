## Why

处理配置目前只存在于单台设备的浏览器存储中。全新安装后用户仍需逐项选择引擎、模型、语言、字体和章节翻译参数，无法直接复用已经验证过的日漫与韩语漫画配置。

## What Changes

- 将当前本机保存的“日漫”和“韩语”配置固化为程序内置处理配置。
- 全新安装默认展示两套配置并选中“日漫”。
- 保留已有安装已持久化的配置，不覆盖、不重复追加内置配置。
- 内置配置不包含 API Key、数据目录或 HTTP 等机器相关设置。

## Capabilities

### New Capabilities
- `built-in-manga-profiles`: 新安装可直接使用的日漫与韩语漫画处理预设。

### Modified Capabilities

## Impact

影响前端处理配置 store 及其测试；不改变后端配置文件、API Key 存储和已有配置迁移规则。
