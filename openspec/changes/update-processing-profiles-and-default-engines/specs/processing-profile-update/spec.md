## ADDED Requirements

### Requirement: 更新当前处理配置
系统 SHALL 允许用户用当前可保存的处理设置覆盖当前选中的处理配置，并 MUST 保留该配置的身份和名称。

#### Scenario: 更新已选中的配置
- **WHEN** 用户应用或保存一个配置、修改处理设置并点击“更新当前配置”
- **THEN** 系统使用当前设置覆盖该配置，后续再次应用时恢复更新后的设置

#### Scenario: 未选中配置
- **WHEN** 当前没有选中的处理配置
- **THEN** 系统禁用“更新当前配置”入口且不修改任何已保存配置

#### Scenario: 其他配置保持不变
- **WHEN** 用户更新当前选中的处理配置
- **THEN** 系统仅替换该配置并保持其他已保存配置不变
