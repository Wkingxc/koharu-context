## Context

主页当前页由 `selectionStore.pageId` 维护，左侧 `Navigator` 已按场景中的页面顺序展示并通过 `setPage` 完成切换。键盘翻页应复用这条状态链路，而不是新增另一套当前页状态。

## Goals / Non-Goals

**Goals:**
- 在主页任意阅读区域使用左右方向键连续翻页。
- 复用现有页面顺序、选中状态和画布刷新逻辑。
- 避免在编辑文本或操作弹窗、表单控件时误翻页。

**Non-Goals:**
- 不新增可配置快捷键。
- 不做首尾循环、自动播放或阅读模式。
- 不改变导航栏页面排序和多选行为。

## Decisions

- 在 `Navigator` 挂载期间监听全局 `keydown`。导航栏折叠时组件仍存在，因此快捷键仍可用于画布阅读。
- 只处理无修饰键的 `ArrowLeft` 与 `ArrowRight`，并调用既有 `setPage`。
- 当事件来自输入、选择、按钮、可编辑区域、弹窗或常见交互控件时忽略该事件。

## Risks / Trade-offs

- [方向键与控件自身交互冲突] → 对可编辑和交互目标进行焦点过滤。
- [到达首尾后行为不明确] → 保持当前页，不循环，避免用户失去阅读位置。
