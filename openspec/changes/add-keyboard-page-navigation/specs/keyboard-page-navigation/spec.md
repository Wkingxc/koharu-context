## ADDED Requirements

### Requirement: 使用左右方向键切换主页图片
系统 SHALL 在主页编辑器中允许用户使用左、右方向键分别切换到当前图片的上一张和下一张，并 MUST 使用现有页面顺序和当前页状态。

#### Scenario: 向后阅读
- **WHEN** 当前图片不是最后一张且用户按下无修饰键的右方向键
- **THEN** 系统切换到下一张图片

#### Scenario: 向前阅读
- **WHEN** 当前图片不是第一张且用户按下无修饰键的左方向键
- **THEN** 系统切换到上一张图片

#### Scenario: 到达章节边界
- **WHEN** 用户在第一张按左方向键或在最后一张按右方向键
- **THEN** 系统保持当前图片且不循环切换

#### Scenario: 用户正在编辑或操作控件
- **WHEN** 键盘事件来自输入框、可编辑区域、按钮、弹窗或其他交互控件
- **THEN** 系统不切换图片并保留控件原有的键盘行为
