## ADDED Requirements

### Requirement: 保存漫画处理配置预设

系统 SHALL 允许用户把当前处理引擎、当前 LLM 目标、目标语言、阅读顺序、默认渲染样式、默认字体、AI 提示词和章节翻译表单参数保存为具名预设。预设 MUST 排除 API Key、数据路径、HTTP、外观和快捷键设置。

#### Scenario: 保存当前配置

- **WHEN** 用户从顶部配置菜单输入唯一名称并确认保存
- **THEN** 系统在本机持久化当前漫画处理配置，并在配置菜单中显示该预设

### Requirement: 从菜单栏切换配置预设

系统 SHALL 在顶部菜单栏提供配置入口，并允许用户一键应用已保存预设。应用时 MUST 同步后端处理引擎与前端 LLM、语言、阅读顺序、默认渲染样式、字体、AI 提示词和章节翻译表单状态。

#### Scenario: 应用已保存预设

- **WHEN** 用户从顶部配置菜单选择一个预设
- **THEN** 系统应用该预设的全部漫画处理设置、加载其 LLM 目标，并标记为当前预设

#### Scenario: 应用失败

- **WHEN** 后端配置更新或 LLM 加载失败
- **THEN** 系统显示错误且不得把该预设标记为当前预设

### Requirement: 管理已保存预设

系统 SHALL 允许用户删除不再需要的预设，且删除预设 MUST 不修改当前已经生效的运行配置。

#### Scenario: 删除预设

- **WHEN** 用户确认删除一个预设
- **THEN** 系统移除该预设；若其原为当前预设则仅清除当前标记
